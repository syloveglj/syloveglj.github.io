Markdown Tutorial

Introduction

Markdown 基本语法

标题

标题使用 # 号来书写，#号之后跟标题 内容。

标题在写的时候，要注意，#号之后一定要跟一个空格，标题与标题之间要有一个空行。

无序列表

Markdown 中的无序列表对应的就是 HTML 中的 ul 标签

  • 在段落中写英文的时候，要注意两边加上空格，为了保持格式清晰。
  • 列表使用一个 - 开头，接着空格，然后是列表的内容。
    • 子列表1
    • 子列表2
      • 野火烧不尽
      • 春风吹又生
    • 子列表3
  • 列表3
  • 列表4

有序列表

有序列表相当于 HTML 中的 ol 标签

  1. 有序列表1
    1. 有序子列表1
    2. 有序子列表2
    3. 有序子列表3
    4. 有序子列表4
  2. 有序列表2
    • 无序列表1
    • 无序列表2
    • 无序列表3

Markdown 总结

  • 标题 #
  • 段落 直接写
  • 无序列表和有序列表
    • 无序列表 -
    • 有序列表 1
  • 粗体和斜体
    • **粗体内容**
    • *斜体内容*
  • 链接和图片
    • [链接的文本](链接的地址)
    • ![图片的alt](图片的链接地址)
  • 文本引用 > 文本引用
  • 代码引用
    • 行内引用 `引用内容`
    • 多行引用 ```多行引用内容```
  • 表格
